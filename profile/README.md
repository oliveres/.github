# 👋 Hi, I'm Oliver

I build small, focused open-source tools that solve problems I actually have.

## Philosophy

- Solve a real problem, then publish — not the other way around
- Document like the next user is also me, six months from now and tired
- Credit the people whose work I built on top of
- Keep it small enough that one person can maintain it

If a project of mine has more commits than stars, that's because I built it
for myself first. If it has more stars than commits, it's probably stable
and I'm onto the next thing.
